getTrialsByRegimeId()

getTrialsByRegimeId(  $rid) : array|null

Get Trials By Regime ID Method

Parameters

$rid

Returns

array|null —

Returns all trials containing a foreign key to the specified regime ID. If there aren't any, then it will return an empty array in any case, otherwise an associative array containing trial objects. Null on fail.
