register()

register(  $ingredients) : string|null

Register Method

Parameters

$ingredients

Returns

string|null —

This is the second part of the register method found in the controller. The ingredients necessary for creating a user are forwarded to this method in an array, which is checked. An SQL query is constructed, as all sanitisation and validation have already been taken care of in the controller, and is executed. If successful, this method actually returns the autoincrement ID of the user it just created in the database, so that additional actions can take place in the controller - predominantly using that ID to finish up changes to the patient table. Null on fail.
